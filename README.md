# Floating Hero Section

A modern and responsive "Legal Work Platform" hero section built using Next.js 14, Tailwind CSS, and Framer Motion.

## 🚀 Features

- Responsive split layout
- Floating animated cards
- Reusable FloatingCard component
- Dark mode support
- Framer Motion animations
- Blurred background effects
- Lucide React icons
- Modern UI design
- Mobile responsive layout

## 🛠️ Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## 📂 Project Structure

```bash
/app
  /components
    FloatingCard.tsx
    HeroSection.tsx
  globals.css
  layout.tsx
  page.tsx
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/floating-hero-section.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
